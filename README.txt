Рассыльщие писем предназначен для отправки писем с помощью сервиса https://sendgrid.com/

Шаблоны писем можно првить с помощью неопсредственно самого сервиса или сторонней платформы, например https://stripo.email/

История рассылки, а также база клиентов ведется в отдельной базе. База построена таким образом, что одному клиенту можно прикрепить несколько Email адресов.

Общий алгоритм работы:
*Программа получается массив шаблонов из Sendgrid. Если шаблон уже использовался в какой-либо рассылке, происходит его запись в собственную БД и, при необходимости, обновление.
*Пользователь выбирает нужный шаблон и заполняет поле со списком адресов. Предусмотрена проверка адресов на верность.
*После отправки если отсутствует клиент(ы) с таким email, создается новый клиент(ы) и ему присваивается этот email/
*Для данного клиента и выбранного шаблона создается запись в истории отправки сообщений.

Код построен по принципу MVC
